                    FOOTBALL DATA GRAPH DATABASE SCHEMA

NODE TYPES:
-----------

1. Country
   └─ Properties: name (unique)

2. Competition  
   └─ Properties: name (unique)
   └─ Relationship: IN_COUNTRY → Country

3. Game
   └─ Properties: gameId (unique), gameDate, homeGoals, awayGoals, winner
   └─ Relationships: 
       ├─ PART_OF → Competition
       ├─ HOME_TEAM → Team
       └─ AWAY_TEAM → Team

4. Team
   └─ Properties: name (unique)

5. Player
   └─ Properties: playerId (unique), name, position, birthDate
   └─ Relationships:
       ├─ PLAYED_IN → Game (with performance stats)
       └─ TEAMMATES → Player (undirected, with gamesTogether count)


RELATIONSHIP TYPES:
-------------------

1. IN_COUNTRY
   Competition ──────────→ Country
   (one-to-one)

2. PART_OF
   Game ────────────────→ Competition
   (many-to-one)

3. HOME_TEAM
   Game ────────────────→ Team
   (many-to-one)

4. AWAY_TEAM
   Game ────────────────→ Team
   (many-to-one)

5. PLAYED_IN
   Player ──────────────→ Game
   Properties: teamName, minutesPlayed, goals, assists, yellowCards, redCards
   (many-to-many)

6. TEAMMATES
   Player ←──────────────→ Player
   Properties: gamesTogether
   (many-to-many, undirected)
